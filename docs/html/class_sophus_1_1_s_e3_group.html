template<typename _Scalar, int _Options>
EIGEN_STRONG_INLINE Scalar* Sophus::SE3Group< _Scalar, _Options >::data ( )
inline
Returns
pointer to internal data

This provides unsafe read/write access to internal data. SE3 is represented by a pair of an SO3 element (4 parameters) and translation vector (three parameters). The user needs to take care of that the quaternion stays normalized.

Note: The first three Scalars represent the imaginary parts, while the forth Scalar represent the real part.

/see normalize()
