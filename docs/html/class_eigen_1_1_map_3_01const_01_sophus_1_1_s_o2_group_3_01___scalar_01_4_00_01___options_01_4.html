Detailed Description

template<typename _Scalar, int _Options>
class Eigen::Map< const Sophus::SO2Group< _Scalar >, _Options >

Specialisation of Eigen::Map for const SO2GroupBase.

Allows us to wrap SO2 Objects around POD array (e.g. external c style complex number)
